W05 Project: Website Planning Document

Site Name

Garden Yield Planner

I chose this project because Reason: it builds on a similar project I did for another class, and a tool that I would like to use someday.

Optional domain name: GardenYieldPlanner.org

Site Purpose

The purpose of the site is to help gardeners calculate how many seeds to plan based on their desired harvest yields. Aligned with the square foot gardening.

Scenarios

What is the yield per one square foot (16 seeds) of tomatos?

WHat is the most common storage method for tomatos?

How many tomatoes per square foot will be grown for the storage method? (example canning)

Color Schema

Red (#F50122), Green (#00A348), Beige (##FFE7C9) Black (#333) and White (#fff)

Reds will be used on man headlines and buttons, and greens on supporting headlines accent areas.

White will be the general background color.

Black will be used to color text.

Beige will be used for an alternate background color.

Typography

A single font family will be used, Source Sans, and will be Headlines will be bold, paragraph text regular. Fonts will be hosted by Google Fonts.
