W05 Project: Website Planning Document

Site Name

Garden Yield Planner

I chose this project because Reason: it builds on a similar project I did for another class, and a tool that I would like to use someday.

Optional domain name: GardenYieldPlanner.org

Site Purpose

The purpose of the site is to help gardeners calculate how many seeds to plant based on their desired harvest yields. Aligned with the square foot gardening.

Scenarios

How many tomato plants should I grow to get enough for canning?

How much can I harvest from one square foot of carrots?

What is the best storage method for the crop I want to grow?

Color Schema

Red (#F50122), Green (#00A348), Beige (#FFE7C9) Black (#333) and White (#fff)

Reds will be used on main headline and buttons, and greens on supporting headlines accent areas.

White will be the general background color.

Black will be used to color text.

Beige will be used for an alternate background color.

Typography

Source Sans 3 will be used for the whole site.

Bold weight will be used for headings.

Regular weight will be used for paragraph text.

Fonts will be loaded from Google Fonts.
